# Examples

Different tech stack implementations for the Azure Project Template.

## Available Examples

### 1. FastAPI + PostgreSQL (Default)
**Path**: `examples/fastapi-postgres/`

Best for:
- Modern Python APIs
- Async operations
- Auto-generated docs
- Type safety

### 2. Flask + MongoDB
**Path**: `examples/flask-mongodb/`

Best for:
- Document databases
- Flexible schemas
- Rapid development
- Simple APIs

### 3. .NET + SQL Server
**Path**: `examples/dotnet-sqlserver/`

Best for:
- Enterprise applications
- Microsoft ecosystem
- Strong typing
- Complex business logic

## How to Use

1. Choose an example
2. Copy to `src/api`
3. Update infrastructure parameters
4. Deploy

## Contributing

Have another stack combination? Submit a PR with:
- Complete implementation
- README with setup instructions
- Infrastructure parameters
- Tests
